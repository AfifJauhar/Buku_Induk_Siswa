✅ Cara Kerja Integrasi
Aksi	Proses
Simpan Data	Data dikirim ke Apps Script → Apps Script menyimpan ke Google Sheets
Upload Foto	Foto dikirim ke Apps Script → Apps Script simpan ke Google Drive → Balikin URL foto
Load Data	Apps Script baca dari Google Sheets → Kirim ke web → Tampilkan di tabel
Edit/Hapus	Apps Script update/delete baris di Google Sheets
Selesai! Sekarang data siswa Anda akan tersimpan otomatis ke Google Sheets, dan foto tersimpan di Google Drive. 🎉
