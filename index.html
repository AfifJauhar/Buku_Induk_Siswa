📘 Panduan Integrasi Google Sheets / Drive (Pihak Ketiga)

🔗 Opsi menyimpan data ke Google Sheets secara otomatis: Anda dapat menghubungkan aplikasi ini dengan Google Sheets menggunakan Google Apps Script. Berikut langkah praktis:

  1. Buat Google Sheet baru, beri nama "Buku_Induk_Siswa".
  2. Buka menu Extensions → Apps Script, tulis kode doPost untuk menerima data JSON (contoh ada di file terpisah).
  3. Deploy sebagai Web App (Execute as Me, Akses Anyone). Salin URL endpoint.
  4. Di aplikasi ini, Anda bisa menambahkan kode fetch ke endpoint setiap kali simpan data. Contoh fungsi syncToSheet() dengan mengirim array students.
  5. Atau gunakan Google Sheets API dengan token OAuth2 untuk pengelolaan penuh.

💡 Tips: Karena aplikasi ini berbasis localStorage, Anda bisa Ekspor manual dengan mencetak sebagai PDF atau menggunakan ekstensi browser. Kami juga menyediakan fungsi ekspor JSON sederhana (bisa ditambahkan).

Jika ingin bantuan pengkodean endpoint Google Apps Script, silakan hubungi admin IT untuk implementasi lanjutan.

Menggunakan Google Drive sebagai backup foto: Foto disimpan sebagai base64 dalam localStorage, namun untuk skalabilitas Anda bisa upload foto ke Drive dan simpan URL-nya. Modifikasi form untuk unggah ke drive menggunakan Picker API.
