📘 Panduan Integrasi Google Sheets / Drive

⚠️ Karena kebijakan CORS Google Apps Script, saat ini aplikasi menggunakan penyimpanan LOKAL (di browser Anda).

📌 Untuk menyimpan data ke Google Sheets secara permanen:
1. Setelah mengisi data, klik "Export JSON" untuk backup data
2. Buka Google Sheets: https://bit.ly/IndukSiswaBook
3. Import file JSON yang sudah diexport

📸 Untuk menyimpan foto ke Google Drive:
Foto akan tersimpan di localStorage browser. Untuk backup permanen, download foto manual.

🔧 Alternatif Solusi CORS:
Jika ingin integrasi real-time, Anda bisa menggunakan layanan seperti corsproxy.io atau deploy Apps Script sebagai Web App dengan konfigurasi yang benar.
